# README Redesign Specification

**Date:** 2026-04-08

**Approach:** Feature-First with Visual Elements

---

## Sections

1. **Header** - Project name, tagline, tech badges (Python, React, FastAPI, Supabase)
2. **What is PETAL** - 2-3 sentence description
3. **Architecture** - ASCII diagram showing agent flow
4. **Features** - Bulleted list of key capabilities
5. **Quick Start** - Copy-paste commands for backend, frontend, Docker
6. **Environment Variables** - Condensed reference
7. **Stack** - Technology table
8. **Project Structure** - Simplified tree view

## Design Principles

- Lead with capabilities, not setup
- Match brutalist aesthetic: bold headers, minimal styling
- Keep commands copy-pasteable
- Show agent architecture visually
- Keep total length under 150 lines
